
# ZenFocus 🧘‍♂️

ZenFocus es una aplicación de productividad minimalista para iOS que diseñe para sesiones de trabajo profundo (Deep Work). El proyecto combina una estética "Glassmorphic" con las últimas APIs de Apple, ocasionalmente la reviso y añado nuevas funciones para probar actualizaciones de apple.

## 🚀 Características Técnicas
*   **Interfaz Declarativa:** Construida totalmente en **SwiftUI**.
*   **Fondo Dinámico:** Implementación de `MeshGradient` para crear un fondo animado fluido que reduce la fatiga visual.
*   **Gestión de Estado:** Utiliza el nuevo framework de observación (`@Observable`) introducido en iOS 17.
*   **Experiencia Táctil:** Integración de Haptic Feedback (`UIImpactFeedbackGenerator`) para una respuesta física premium.
*   **Arquitectura:** Patrón **MVVM** para una clara separación entre la lógica del temporizador y la interfaz.

## 🛠️ Requisitos
*   Xcode 16.0+
*   iOS 17.0+
